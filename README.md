## NoteLogic

NoteLogic é um aplicativo de lista de tarefas (ToDo list) desenvolvido com Django.

### Funcionalidades ✨

- Cadastro de usuário
- Autenticação de usuário (login/logout)
- Adição de tarefas
- Marcar tarefas como concluídas ou pendentes
- Remoção de tarefas

### Tecnologias Utilizadas 🛠️

- Django
- HTML
- CSS
- FontAwesome
